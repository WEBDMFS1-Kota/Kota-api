# Kota-api

## Getting started

Install nodejs 18.2.0
Install docker
Run the database with:

```
docker-compose up -d
```

Install node modules with

```bash
npm i
```

then start the server with 

```bash
npm run dev
```

## Prisma

Create a .env file with the .env.template

Run `npx prisma generate`

Use prisma studio with

```bash
npx prisma studio
```
